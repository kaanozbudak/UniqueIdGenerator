- Mongo

- Postman



Make sure use MakeFile


Installation:
- Start with setup MongoDB
- Pull Mongodb 
- _`docker pull mongo`_
- Check is it installed
- _`docker image ls`_
- Run mongodb as daemon mode
- _`docker run -d -p 27017:27017 mongo:latest`_
- Check is mongodb up
- _`docker ps`_
